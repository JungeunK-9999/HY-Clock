# 대학 연합 종강시계 Extension

`Not in progress yet`


## 프로젝트 시작하기

1. `git clone https://github.com/Neulhan/UnivClock.git`
2. [chrome://extensions/](chrome://extensions/) 접속
3. `압축해제된 확장 프로그램을 로드합니다.` 클릭
4. 클론받은 UnivClock 디렉토리 선택
